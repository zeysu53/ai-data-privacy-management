# AI-Supported Personal Data and Digital Communication Management System

## Project Overview
This project focuses on the problem of continuing promotional and marketing communications after users terminate their membership on digital platforms.

## Purpose
The main purpose of this project is to enhance users' control over their personal data and to reduce legal and operational risks for companies within the scope of data protection regulations.

## Scope
- Personal data management processes
- Digital communication control mechanisms
- AI-supported system design
- Data protection law (KVKK) analysis

## Methodology
- Analysis of legal and technical aspects of personal data processing
- Identification of required data sources and risks
- Conceptual design of an AI-supported solution
- Evaluation of ethical and data security considerations

## Outcomes
- Prevention of unlawful personal data processing
- Improved user trust and satisfaction
- Proposal of an automated and traceable data management system
